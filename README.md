# ETS2 Cargo Dispatcher PRO 🚛

Sistema profissional de despacho de cargas para Euro Truck Simulator 2, com suporte a descriptografia nativa de saves e suporte ao Mapa RRB (Brasil).

## 🚀 Funcionalidades
- Descriptografia nativa de arquivos `game.sii` (ScsC v3).
- Scanner binário universal de cidades (Base + DLCs + Mods).
- Interface Premium em Flet com modo escuro.
- Randomização inteligente de rotas e cargas.

## 🛠️ Tecnologias
- Python 3.12+
- Flet (UI)
- Cryptography (AES-256-CBC)

## 📦 Como Instalar
```bash
pip install -r requirements.txt
```

## 🎮 Como Usar
```bash
python -m app.main
```
